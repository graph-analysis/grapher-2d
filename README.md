# 图前端可视分析APP

基于 qiankun 微应用的单页插件，用于进行大数据图可视分析

## 效果预览

![image](https://user-images.githubusercontent.com/53158137/126269297-6f2f75cb-4d79-45fc-bfd5-caf1cdf94c8e.png)

## Getting Started

Install dependencies,

```bash
$ pnpm i 
```

Start the dev server,

```bash
$ pnpm dev
```

release a publish,

```bash
$ pnpm release
```
